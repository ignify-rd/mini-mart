# OrderWorder — Mini Mart nội bộ

Ứng dụng đặt hàng cửa hàng tiện lợi / mini mart: khách xem menu, thanh toán VietQR; admin quản lý đơn, tồn kho, thực đơn và danh mục.

**Stack:** Next.js · Google Sheets · NextAuth · SWR · bcrypt · xtreme-ui

## Chạy local

```bash
pnpm install
# điền .env (xem docs/ENVIRONMENT.md và docs/GOOGLE_SHEETS_SETUP.md)
pnpm play
```
